CS50 Commerce

A simple Commerce project built with Django as part of the CS50 Web course.

Project Overview

This project allows users to browse products, view details, add items to a shopping cart, and manage orders. It is built using Django and includes multiple apps to handle products, orders, and user authentication.

Technologies

Python 3, Django, HTML/CSS, SQLite (default database)

Setup & Run Locally
Clone the repository, navigate to the project folder, optionally create a virtual environment, install dependencies (or Django manually), run migrations, and start the development server. Then open your browser at http://127.0.0.1:8000/
 to see the project in action.

Notes

This project is backend-based (Django), so GitHub Pages cannot host it live.

All code and structure can be viewed directly on GitHub.
